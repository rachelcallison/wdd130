ICE Challenges

Prophet Card

This project demonstrates my understanding of CSS layout and styling techniques. I used CSS Grid to structure the page and center the main content, along with custom colors, typography, and spacing to create a cohesive design. I applied borders, border-radius, and padding to build a styled card component, and incorporated image styling for visual balance. This project helped me practice organizing content, controlling layout placement, and creating a clean, visually appealing webpage using HTML and CSS.

Apostle Spotlight

This project demonstrates my ability to use CSS Grid to organize complex layouts and structure content into responsive sections. I created grid-based layouts for different groups, used repeat() and fractional units to evenly space items, and centered content using alignment properties. I also styled circular images using border-radius: 50% and implemented hover effects to add interactivity. Additionally, I practiced using background images, spacing, and typography to create a visually cohesive and polished design. This project helped me strengthen my layout skills and better understand how to combine structure and styling in CSS.

Valentines

This project demonstrates my ability to enhance visual design using custom fonts, typography, and styling effects. I imported and applied Google Fonts to create a unique theme and experimented with text shadows to add depth and emphasis to the design. I also used absolute positioning and transform properties to center text over an image, creating a layered visual effect. This project helped me practice combining layout techniques with creative styling to build a visually engaging webpage.

Favortite Devo

This project demonstrates my ability to create organized and user-friendly content sections using CSS. I styled tables with borders, spacing, and consistent alignment to clearly display data, making it easy to read and visually structured. I also designed custom input boxes with background colors, padding, and box shadows to highlight important content and create a polished, interactive feel. This project helped me practice combining layout, typography, and component styling to build a clean webpage.

Flags

This project demonstrates my ability to use CSS Grid to create complex layouts and visually replicate real-world designs. I practiced building structured grids to recreate flags and patterns, carefully placing colors and shapes in specific grid cells. This project helped me develop skills in positioning, layering, and organizing content in a grid, as well as applying color and shape styling to create accurate and visually appealing designs.

Positioning Exercises

This project demonstrates my ability to use CSS Grid and positioning techniques to create structured, multi-section layouts. I practiced arranging elements of various sizes and colors across multiple grid configurations, including nested grids and overlapping content. The project also helped me apply floats, z-index, and responsive spacing to control how content is displayed, as well as practice styling interactive elements like hints with hover and checkbox-triggered displays. Overall, it strengthened my skills in layout, alignment, and organizing complex visual structures using CSS.
